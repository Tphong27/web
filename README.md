# Ôn Tập Chủ Nghĩa Xã Hội Khoa Học

## Giới thiệu
Trang web ôn tập CNXHKH dành cho sinh viên FPT University. Cung cấp tài liệu học tập toàn diện với nội dung dễ hiểu, bám sát giáo trình và slide bài giảng.

## Cấu trúc trang web

Trang web bao gồm 4 trang riêng biệt:

| Trang | File | Mô tả |
|-------|------|--------|
| Trang chủ | `index.html` | Giới thiệu tổng quan, mục lục các chương |
| Nội dung ôn tập | `content.html` | 7 chương kiến thức đầy đủ |
| Ngân hàng câu hỏi | `questions.html` | Câu hỏi lý thuyết có đáp án |
| Quiz | `quiz.html` | Bài kiểm tra trắc nghiệm |

## Các tính năng

- **Trang chủ**: Tổng quan với thống kê số chương, số câu hỏi, số quiz
- **Nội dung ôn tập**: 
  - 7 chương theo cấu trúc giáo trình
  - Điều hướng nhanh theo chương
  - Trình bày rõ ràng với các tiêu đề, danh sách, bảng biểu
- **Ngân hàng câu hỏi**:
  - Bộ lọc theo chương
  - Toggle hiển thị đáp án
  - Câu hỏi lý thuyết dạng tự luận
- **Quiz**:
  - Chọn chủ đề từ các chương
  - Câu hỏi trắc nghiệm đa lựa chọn
  - Theo dõi tiến độ và điểm số
  - Xem lại đáp án sau khi hoàn thành

## Hướng dẫn sử dụng

1. Mở `index.html` trong trình duyệt để bắt đầu
2. Sử dụng navigation bar để chuyển giữa các trang
3. Trên mobile, nhấn menu icon để mở navigation
